# Homebrew Tap for Verzio

Official [Homebrew](https://brew.sh) cask for [Verzio](https://verzio.pro) -- your home for Mac updates.

## Installation

First, add the tap:

```sh
brew tap jmpellizzer/verzio
```

Then install Verzio:

```sh
brew install --cask verzio
```

## Updating

```sh
brew upgrade --cask verzio
```

## Uninstalling

```sh
brew uninstall --cask verzio
```

## Requirements

- macOS Sonoma (14.0) or later

## Learn More

Visit [verzio.pro](https://verzio.pro) for more information.
